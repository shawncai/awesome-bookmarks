# 多余逗号引发的错误

2018-07-12

刚入前端的时候看错误日志，ie 的错误日志特别多，一直没找到原因，后来发现是 JSON 最后一组键值后多逗号。

// 所有浏览器都正常
var json_normal = {
  id: 1,
  name: "John"
};

// ie 报错，其它游览器正常
var json_error = {
  id: 1,
  name: "John",
};

好在现在有了 eslint 或者 preitter这种工具，这种错误很少会再发生了。
