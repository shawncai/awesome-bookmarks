# 函数变量必填校验

2018-11-02

这里分享一个平时写 ES6 的时候一个小技巧。如何简单的校验并强制在使用这个函数时必须传参数。

const isRequired = () => {
  throw new Error('Missing parameter')
}

const foo = (something = isRequired()) => {
  console.log(something)
  return something
}
foo(123)
foo() // Error:Missing parameter
